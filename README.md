## Simple Go HTTP server

Simple HTTP server that doesn't do anything else aside from showing in logs requests received.

### Example

From one terminal:

```shell
$ make run
go build -o simple-http-server -v
./simple-http-server
2023/05/09 12:36:00 Server listening on 127.0.0.1:8080...
```
